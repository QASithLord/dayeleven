Positioning Properties

HTML elements by default, take up the full width of the page. We can keep elements from sitting on top of each other by using positioning properties. There are four possible values.

  1. block: will not let anything sit next to it on the page, and takes up the full width. It makes the element a block box.
  2. inline-block: this also makes the element a block box, but will allow other elements to sit next to it on the same line.
  3. inline: makes the element sit on the same line as another element. It will not format it like a block. Additionally, it only tkaes up as much width as needed (not the whole line). Best suited for HTML elements that are blocks by default, such as headers and paragraphs.
  4. none: This will make the element and its content disappear from the page
